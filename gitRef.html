Git Cheat Sheet

git Clone (website/file address) This command is needed only once to download the file which you will be working on. It will download and set up the files, based on the folder structure in gitHub.

dir This command shows the directory/files in the folder.

dir /ah This command will show ALL the files, even the hidden ones.

git Pull (website/file address) This command pulls down the current files from gitHub.

git Status This command shows the changes you made on your localhost.

git add . This command updates and saves the changes made on your localhost.

git Commit -m"This is the comment section" This command is used to comment the changes and what was updated, it is key to enter in a detailed account of your revisions. This will help you and your team track what was added and/or modified.

git Push The command pushes or adds your changes to the gitHub file, creating a new updated file. This file is also carries your comments on the additions you made.
